Place your app icon here as: icon.ico

Requirements:
  - File name: icon.ico
  - Recommended size: 256x256 px (multi-size .ico with 16, 32, 48, 256)

Free tools to create .ico files:
  - https://icoconvert.com  (upload a PNG, download .ico)
  - https://convertico.com

If you skip this, electron-builder will use the default Electron icon.
To skip: remove the "installerIcon" and "uninstallerIcon" lines from package.json.
