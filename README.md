# Resell Tracker

A desktop app for resellers to track sales, inventory, and packages — all in one place.

## Features

- **Sales tracking** — log every sale with platform, purchase price, sale price, fees, and auto-calculated profit
- **Monthly chart** — visualize revenue, spending, and profit by month with toggleable series
- **Inventory management** — track your stock with photos, quantity, store, buy price, and estimated resell value
- **Package tracking** — monitor shipments with carrier auto-detection and live status updates
- **Dark mode** — toggle a neon dark theme from settings
- **Discord auth** — access is gated behind a Discord subscriber role, verified server-side on every launch

## Installation

Download the latest `.exe` installer from the [Releases](../../releases) page and run it.

The app will notify you automatically when a new update is available.
